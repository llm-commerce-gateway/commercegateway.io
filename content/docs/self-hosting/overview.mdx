## System requirements

- Node.js >= 18
- 512MB RAM minimum
- Linux, macOS, or Windows host

## Installation options

- `npx` zero-install run
- npm/pnpm local install
- Docker image deployment

## Configuration

Use environment variables for ports, auth, provider keys, and backend connectors.

Full reference: [Configuration](/docs/self-hosting/configuration)
