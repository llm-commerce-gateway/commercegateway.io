## Who hosts Commerce Gateway?

Commerce Gateway is created and maintained by Better Data. The hosted runtime runs on Better Data infrastructure.

## What's included in hosted

- ✓ Multi-tenant gateway with brand isolation
- ✓ Brand portal for merchant API configuration
- ✓ Usage analytics per brand and adapter
- ✓ Commerce Gateway Registry for LLM discovery
- ✓ Managed SSL, uptime SLA, and dedicated support
- ✓ Stripe billing integration
- ✓ Okta SSO for enterprise teams

## Open source commitment

The gateway protocol, tool schema, and self-hostable server are MIT licensed and remain open source. The hosted platform is a commercial service built on top.

Comparable model:

- Nginx (OSS) vs Nginx Plus (commercial)
- Kubernetes (OSS) vs GKE/EKS/AKS (hosted)
- Commerce Gateway (OSS) vs Better Data hosted platform (commercial)
