## What the hosted platform includes

### Managed gateway infrastructure

Better Data runs Commerce Gateway on managed infrastructure with automatic updates, SSL, and uptime monitoring. No servers to manage, no deployments to coordinate.

### Gateway Console

The operator UI for managing providers, connectors, keys, and registry status. Free tier available for self-hosted deployments. Fully managed on the hosted platform.

### Commerce Registry

Your gateway is listed and verified in the public Commerce Registry at `registry.betterdata.co`. Domain verification and GTIN product-line claims are managed through the platform.

### Brand portal

Merchant-facing configuration UI for brands connecting their commerce backends — Shopify, BigCommerce, WooCommerce, or custom API. Each brand gets isolated configuration and credentials.

### Usage analytics

Full dashboard showing tool call volume, adapter usage, connector sync status, error rates, and latency by brand and time period.

### Registry MCP

Your gateway is reachable via the Commerce Registry MCP at `mcp.commercegateway.io` — enabling LLM clients that support MCP to discover and connect to your gateway without manual configuration.

### Enterprise features

- Okta SSO and SCIM provisioning
- Stripe billing integration
- Multi-tenant brand isolation
- Dedicated support and SLA agreement

## Open source commitment

The gateway protocol, tool schema, and self-hostable packages are **MIT licensed** and remain open source. The hosted platform is a commercial service built on top.

The same pattern as:

- Nginx (OSS) vs Nginx Plus (commercial)
- Kubernetes (OSS) vs GKE / EKS / AKS (hosted)
- Commerce Gateway (OSS) vs Better Data hosted platform (commercial)
